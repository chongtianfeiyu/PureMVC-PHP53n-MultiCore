Class View

View

public class View

A Multiton IView implementation. In PureMVC, the View class assumes these responsibilities: - Maintain a cache of IMediator instances. - Provide methods for registering, retrieving, and removing IMediators. - Notifiying IMediators when they are registered or removed. - Managing the observer lists for each INotification in the application. - Providing a method for attaching IObservers to an INotification's observer list. - Providing a method for broadcasting an INotification. - Notifying the IObservers of a given INotification when it broadcast.

registerMediator

public void registerMediator(IMediator mediator)

Register Mediator Register an IMediator instance with the View. Registers the IMediator so that it can be retrieved by name, and further interrogates the IMediator for its INotification interests. If the IMediator returns any INotification names to be notified about, an Observer is created encapsulating the IMediator instance's handleNotification method and registering it as an Observer for all INotifications the IMediator is interested in.

Parameters:
mediator - Reference to the IMediator instance.
